API Basejump: Image Seach API

User stories:
    1) I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
    2) I can paginate through the responses by adding a ?page=2 parameter to the URL.
    3) I can get a list of the most recently submitted search strings.

Example usage:

    1) URL/api/imagesearch/cars
    2) URL/api/imagesearch/cars?page=2
    3) URL/api/latestsearches
